Changelog

All notable changes to the Test Equipment Matrix project
v2.8.0 2026-04-10
Added
File Attachments on Flow Nodes
  • New "Attachments" section in flow.html activity detail popup (below Equipment Items)
  • Paperclip "Add" button opens file picker — any file type, multiple files, up to 10 MB each
  • Click a filename to open the file directly in a new browser tab
  • File-type icons: 📄 PDF, 🖼 image, 📊 spreadsheet, 📝 document, 📦 archive, 📎 other
Changed
Sticky Scrollbar & Header (index.html)
  • Horizontal scrollbar always visible within viewport — no more scrolling to page bottom
  • Table header rows pinned with position: sticky while scrolling vertically
  • Freeze mode columns retain higher z-index and work correctly on top of sticky headers
Changed
Location Colors Sync to Gantt Chart
  • Gantt chart reloads custom location colors in real-time when changed on index.html settings
  • Bars, map nodes, transport paths, legend dots all use single source of truth
v2.7.0 2026-04-10
Fixed
Edge Port Indicators Clickable (flow.html)
  • Port dots now have pointer-events: all and click handlers
  • Click any port dot on a node to snap edge endpoint — no dragging needed
Changed
Bidirectional Location & Date Sync
  • TestManager.updateLocation() and updateDate() now sync to gantt bars via _syncToGanttBars()
  • Gantt chart listens for cross-tab changes and re-renders when matrix page writes
v2.6.0 2026-04-10
Added
Edge Line Repositioning (flow.html)
  • Click an edge to select — cyan highlight, draggable handles at both endpoints
  • Port indicator dots on all 4 sides of connected nodes
  • Drag handle or click port dot to reroute edge — snaps to nearest port
  • Custom fromSide/toSide saved per edge, persists across reloads
  • Right-click → "Reset Line Ports" clears custom routing
  • Bezier paths and arrowheads support all 16 side combinations
v2.5.0 2026-04-10
Fixed
Locked Filters Highlight on Flow Page Revisit
  • _renderOnce() now calls _applyWpFilter() — locked WP filters highlight nodes immediately
Changed
Separated Locked Filters Between Pages
  • index.html uses matrixLockedTypeFilters / matrixLockedWpFilters
  • flow.html uses flowLockedWpFilters
  • Backwards-compatible migration from old shared keys
Fixed
Scroll Jump When Adding Test Activity
  • Removed forced horizontal scroll — new columns respect current filter state
v2.4.0 2026-04-10
Fixed
Gantt Bar Dragging After Assigning Dates
  • Rewrote _rebuildTransfers() — bars can freely overlap, dates never forcibly rewritten
  • Transfers only inserted when genuine gap exists between consecutive bars
v2.3.0 2026-04-10
Changed
Bar Right-Click: "Change Location" Popup
  • Replaced inline location list with dedicated modal popup
  • Context menu now cleaner: Edit Dates, Change Location, Test Activities, Delete Bar
Changed
Map Location Click → Rich Info
  • Left-click location nodes (SVG or Leaflet) shows equipment + their linked test activities
  • Interactive tooltip with smart positioning, scrollable, dismisses on click outside
v2.2.0 2026-04-10
Added
Test Activities Without Dates on Gantt Chart
  • Tests without dates now open Add modal pre-linked with purple badge

Changed
Bidirectional Test Data Sync & Timeline Alignment
  • _syncTestData() syncs location in addition to dates
  • Timeline slider restructured to align with gantt bars (220px label column)

Verified
